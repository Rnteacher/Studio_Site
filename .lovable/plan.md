

# סטודיו דוריאן — Creative Youth Studio Website

## Overview
A modern, RTL Hebrew website for a youth-led creative studio from "החממה" high school in Hod Hasharon. The site showcases student talents and services offered to NGOs and small businesses.

## Design System
- **Warm, artistic palette**: Cream background (#FFECCC), deep magenta headings (#870058), red buttons (#A4303F), soft green content sections (#C8D6AF), warm tags (#F2D0A4)
- **Style**: Clean, spacious, rounded cards, subtle shadows, creative icons — youthful but professional
- **Fully RTL layout** with modern Hebrew typography

---

## Pages & Features

### 1. Homepage
- **Hero Section**: Large artistic heading "סטודיו דוריאן" with subheading and two CTA buttons (About / Contact). Subtle creative background illustrations
- **Search & Filter Section**: Live search bar + hierarchical tag filtering system with 11 main categories (Art, Music, Sewing, Photography, etc.) that expand into sub-tags on click. Multiple tag selection supported, with visible active filters
- **Student Cards Grid**: Responsive grid (4/2/1 columns) showing student photo, name, short description, and 3 service tags. Clicking opens their profile page

### 2. Student Profile Page (Dynamic Route)
- Large image with name and tagline
- Full description section
- Services grouped by category
- Contact section with email, phone, social links, and a contact form
- Soft green (#C8D6AF) background sections for visual rhythm

### 3. About Page
- Studio mission and story (youth initiative from החממה high school)
- Values: Community, Responsibility, Professionalism, Young Entrepreneurship
- Image placeholders for behind-the-scenes moments

### 4. Contact Page
- Contact form: Name, Email, Phone, Requested Service, Message
- Studio email display
- Clean, inviting layout

---

## Data & Content
- 11 students pre-populated with real service data (מיאל, אמה, מוריה, נוגה, צבי, עידו, יערה, אלון, בנימין, עידן, מאיה)
- Structured data model with categories, services, and contact info per student
- Full category/sub-tag taxonomy across all 11 categories

## Key Functionality
- Live search filtering across students and services
- Multi-tag hierarchical filtering with visual feedback
- Dynamic routing for individual student profiles
- Smooth fade/hover animations
- Fully responsive and accessible
- Scalable structure ready for future additions (testimonials, portfolio, new students)
